## Para saber mais: dados x informação

"Dados" e "Informação" na linguagem cotidiana podem parecer sinônimos, mas em Tecnologia da Informação, elas não são, apesar de ser uma diferença sutil! Mas entender esses conceitos é fundamental para compreender como a tecnologia manipula informações.

Enquanto **dados são elementos brutos, como números ou texto sem significado imediato**, a **informação é o que obtemos quando esses dados são organizados, processados e interpretados, gerando conhecimento útil**. Vamos nos aprofundar mais nesse conceito!

Imagine que você recebeu um arquivo cheio de números aleatórios, como "28, 18, 7, 32". Sozinhos, esses números são apenas dados, não dizem nada específico. Agora, se alguém te explicar que esses números representam as temperaturas diárias de uma cidade durante a semana (sua cidade também varia de temperatura assim?), isso se torna informação — útil e significativa. **Os dados são o ponto de partida, mas a informação é o destino**.

No exemplo do nosso projeto, os dados "Carla" e 200 , podem não significar nada isoladamente, mas se eu digo que "O saldo da conta de Carla é R$200,00", acabo gerando uma informação através dos dados.

**Dados são como ingredientes crus, enquanto informação é o prato pronto** e saboroso que você obtém quando sabe como combiná-los.
